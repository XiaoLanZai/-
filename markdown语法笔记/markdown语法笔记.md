# 列表

列表分为无序列表和有序列表

* 无序列表，使用 `*`、 `+` 、`-` ，再加一个空格作为列表的标记
* 有序列表，使用数字并加上`.`号，再加一个空格作为列表的标记

```
* 无序列表 1

* 无序列表 2

* 无序列表 3

1. 有序列表 1

2. 有序列表 2

3. 有序列表 3
```

列表可以嵌套 `Ctrl`+`]` 缩进
